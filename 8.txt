Step 1: Create a MongoDB Collection

use myblogdb;


db.createCollection("articles");

Step 2: Insert Documents (Create - CRUD)


db.articles.insertOne({
    title: "Introduction to MongoDB",
    content: "MongoDB is a NoSQL database...",
    author: "John Doe",
    author_age: 30,
    category: "Database",
    comments: [
        { name: "Alice", remarks: "Great article!" },
        { name: "Bob", remarks: "I learned a lot." }
    ]
});

db.articles.insertOne({
    title: "Getting Started with Node.js",
    content: "Node.js is a JavaScript runtime...",
    author: "Jane Smith",
    author_age: 25,
    category: "Web Development",
    comments: [
        { name: "Charlie", remarks: "Very informative." }
    ]
});

Step 3: Display Documents (Read - CRUD)



db.articles.find();


db.articles.find({ category: "Database" });

Step 4: Update Documents (Update - CRUD)

db.articles.updateOne(
    { title: "Introduction to MongoDB" },
    { $set: { content: "MongoDB is a powerful NoSQL database..." } }
);

Step 5: Delete Documents (Delete - CRUD)

db.articles.deleteOne({ title: "Getting Started with Node.js" });
